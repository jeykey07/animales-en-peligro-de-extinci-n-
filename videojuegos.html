CREACION DE VIDEOJUEGOS

Diferentes plataformas para crear diferentes tipos de videojuegos

Godot Engine

Especialmente GDScript, aunque también es compatible con lenguaje C# y C++

Godot Engine es un motor de videojuegos repleto de características, multiplataforma para crear juegos 2D y 3D por medio de una interfaz unificada. Provee un conjunto exhaustivo de herramientas comunes, para que los usuarios puedan enfocarse en crear juegos sin tener que reinventar la rueda. Los juegos pueden exportarse en un sólo clic a numerosas plataformas, incluyendo las principales plataformas de escritorio (Linux, macOS, Windows), plataformas móviles (Android, iOS), así como plataformas y consolas basadas en la web.

Godot es completamente gratis y de código abierto bajo la permisiva licencia MIT (Licencia del Instituto Tecnológico de Massachusetts). Sin condiciones, sin regalías, nada. Los juegos de los usuarios son suyos, hasta la última línea de código del motor. El desarrollo de Godot es totalmente independiente y dirigido por la comunidad, lo que permite a los usuarios ayudar a dar forma a su motor para que coincida con sus expectativas. Está respaldado por la Godot Foundation (Fundación Godot) sin fines de lucro.

VENTAJAS:

  • Gratis.
  • Editor potente con una interfaz visual e intuitiva.
  • Actualizado constantemente.
  • Comunidad excelente.
  • Permite exportar juegos a versiones de PC, celulares y web (HTML5).

    • DESVENTAJAS:

  • Físicas pobres al crear juegos en 2D.
  • Terminología del programa algo confusa.

    • RPG Maker

    Lenguaje C++, Ruby, C++ y Javascript

    RPG Maker es uno de los programas clásicos más conocidos cuando se trata de crear juegos. Como indica su nombre, se enfoca en la creación de RPGs en 2D y posee herramientas y recursos innumerables para ese género. Se trata de un programa especializado en esa área, aunque, con un poco de creatividad, permite crear juegos de otros estilos.

    RPG Maker posee una interfaz muy intuitiva y no exige conocimientos de programación. Eso sí, permite que los usuarios más avanzados programen todo tipo de códigos. Como programa de creación de juegos, es indicado para PC, navegador web y Android.

    VENTAJAS:

  • Interfaz muy intuitiva.
  • Códigos simplificados, permitiendo a su vez una programación más avanzada.
  • Comunidad inmensa y activa.
  • Tiene versiones de prueba y un extenso contenido con recursos adicionales.

    • DESVENTAJAS:

  • Compatibilidad algo limitada para los juegos.
  • No es muy indicado para los principiantes en creación de juegos.
  • Limitación en géneros de juego.
  • Programa caro a menos que sepas crear tus propios recursos.

    • Unity

    C#

    Unity es uno de los programas para hacer videojuegos más famosos y accesibles para los desarrolladores. Es utilizado tanto por empresas como por personas que crean juegos por hobby. Unity es una plataforma completa para desarrollar videojuegos con un editor visual e intuitivo.

    Unity es un motor gráfico que soporta gráficos 2D y 3D. Proporciona un ecosistema completo con muchos recursos, tutoriales y una tienda en línea donde encontrar complementos, recursos visuales, efectos sonoros y mucho más.

    VENTAJAS:

  • Plataforma multiplataforma: permite exportar a diferentes plataformas.
  • Editor potente con muchas herramientas de trabajo.
  • Recursos abundantes y enorme comunidad.
  • Versión gratuita muy completa.
  • Assets Store con miles de recursos.

    • DESVENTAJAS:

  • Requiere conocimientos en C#.
  • Los gráficos de alta calidad requieren conocimientos avanzados.
  • Cambios en su modelo de licencias y costos.

    • Unreal Engine

    C++ y Blueprints (programación visual)

    Unreal Engine es un motor gráfico de alta calidad y potencia creado por Epic Games. Está fuertemente orientado a la creación de videojuegos con gráficos 3D fotorrealistas. Su sistema de programación visual llamado Blueprints permite crear juegos sin escribir código tradicional.

    Este motor gráfico es ampliamente utilizado por estudios profesionales para crear juegos AAA, aunque su versión pública está disponible para cualquier desarrollador. También se utiliza en la industria del cine y la televisión para crear efectos visuales.

    VENTAJAS:

  • Gráficos de alta calidad y fotorrealistas.
  • Sistema Blueprints para programación visual.
  • Editor potente con muchas opciones avanzadas.
  • Comunidad grande y soporte directo.
  • Completamente gratuito hasta que el juego genera ingresos.

    • DESVENTAJAS:

  • Curva de aprendizaje pronunciada.
  • Requisitos de hardware elevados.
  • Poco optimizado para juegos 2D.
  • Complejidad general del motor.

    • GameMaker Studio

    GML (GameMaker Language) y programación visual

    GameMaker Studio es una plataforma de desarrollo específicamente diseñada para crear juegos 2D. Está orientada tanto a principiantes como a profesionales, con un sistema de programación visual y su propio lenguaje llamado GML.

    Este software está optimizado para que la creación de prototipos y juegos completos sea rápida y eficiente. Ha sido utilizado para crear éxitos comerciales como Hotline Miami, Undertale y muchos otros.

    VENTAJAS:

  • Excelente para juegos 2D.
  • Interfaz intuitiva y fácil de usar.
  • Sistema DnD (Drag and Drop) para aprender sin programar.
  • Exportación a múltiples plataformas.
  • Comunidad activa y muchos tutoriales disponibles.

    • DESVENTAJAS:

  • Limitado para juegos 3D.
  • Licencias de pago para funcionalidades avanzadas.
  • GML es un lenguaje específico que no se usa fuera de GameMaker.
  • Menos flexible que motores completos como Unity.

    • Construct

    Sin programación (visual) y JavaScript

    Construct es un motor para crear juegos 2D sin necesidad de programar. Utiliza un sistema de eventos y condiciones que resulta muy intuitivo para principiantes y creadores sin experiencia técnica previa.

    Esta herramienta se ejecuta en el navegador y permite crear juegos HTML5 que funcionan en múltiples plataformas. Es ideal para prototipar ideas rápidamente y crear juegos sencillos.

    VENTAJAS:

  • No requiere conocimientos de programación.
  • Ideal para principiantes absolutos.
  • Interfaz muy intuitiva basada en eventos.
  • Exporta principalmente a HTML5 y plataformas móviles.
  • Editor basado en navegador, accesible desde cualquier lugar.

    • DESVENTAJAS:

  • Limitado a juegos 2D.
  • Versión gratuita con funciones limitadas.
  • Menor rendimiento que otros motores.
  • Menos flexible para proyectos complejos.

    • Buildbox

    Sin programación

    Buildbox es una herramienta para crear juegos sin código destinada principalmente a dispositivos móviles. Su filosofía es permitir que cualquier persona, independientemente de su conocimiento técnico, pueda crear juegos atractivos arrastrando y soltando elementos.

    Esta plataforma es conocida por su rapidez al crear prototipos y por haber sido utilizada para desarrollar algunos juegos exitosos en tiendas móviles. Es ideal para crear juegos arcade o de estilo casual.

    VENTAJAS:

  • No requiere ningún tipo de programación.
  • Interfaz de arrastrar y soltar.
  • Ideal para juegos móviles.
  • Creación rápida de prototipos.
  • Buenos resultados visuales sin mucho esfuerzo.

    • DESVENTAJAS:

  • Opciones limitadas de personalización.
  • Precio elevado comparado con otras opciones.
  • Más restrictivo en tipos de juego.
  • Menos flexible para ideas originales.

    • Stencyl

    Visual por bloques (basado en Scratch), Haxe (lenguaje similar a JavaScript/C++):

    Stencyl es un término medio entre Scratch y Construct 3. El programa también presenta un aspecto educativo centrado en enseñar lo básico de programación, mas permite una variedad mucho mayor de juegos y herramientas que Scratch.

    Stencyl es perfecto para juegos Flash. De hecho, bastante de los juegos creados en Stencyl se encuentran en páginas como ArmorGames, Kongregate y Newgrounds. Es un programa muy bueno para crear juegos de plataformas, defensa de torres, arcade y RPGs en 2D.

    VENTAJAS:

  • Buena compatibilidad con Flash, Windows, Mac, Linux, iOS, Android y HTML5.
  • Enseña conceptos básicos de programación.
  • Es posible obtener patrocinadores para los juegos.
  • Versión gratuita de uso personal y que te permite crear juegos Flash.

    • DESVENTAJAS:

  • Necesitas pagar suscripción anual para aprovechar todas las funcionalidades y eliminar la marca de agua.
  • No posee tantas herramientas.
  • Aunque compatible, no es ideal para crear juegos para celulares.

    • Roblox Studio

    Lua

    Roblox Studio, como ya imaginarás, es el programa utilizado para crear juegos en Roblox. Ofrece todas las herramientas necesarias para crear juegos en 3D con una rapidez y sencillez sorprendentes. Cuenta con distintas funcionalidades y modelos listos para arrastrar y colocar a tu voluntad.

    Los juegos creados en Roblox Studio siguen las mismas mecánicas clásicas de Roblox. Aun así, tú tienes el control completo en la creación de escenario y reglas de juego.

    VENTAJAS:

  • Gratis.
  • Creación de juegos fácil y totalmente en 3D.
  • Muchísimo contenido y una comunidad potente, siempre activa.

    • DESVENTAJAS:

  • Compatibilidad muy baja, pues los juegos solo funcionan en Roblox.
  • El programa solo corre en Windows y Mac.

    • GDevelop

    Programación por eventos visuales (sin escribir código), Internamente usa JavaScript

    GDevelop es uno de los programas de creación de juegos que mayores compatibilidades posee para diferentes plataformas. Queremos remarcar su potente sistema de eventos, el cual te permite determinar muchos aspectos del juego con suma sencillez y sin necesidad de pelear con códigos.

    Es totalmente gratis y, aunque tiene menor documentación y actividad respecto a otros programas, GDevelop es un buen software si deseas crear un juego simple en 2D.

    VENTAJAS:

  • Gratis
  • Es de código abierto, lo que permite una personalización alta.
  • Sistema potente de eventos.
  • Juegos compatibles con múltiples plataformas de PC, móviles y páginas webs.

    • DESVENTAJAS:

  • Lento
  • Comunidad no muy activa y documentación insuficiente.
